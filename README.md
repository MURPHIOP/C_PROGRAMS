# 📘 DSA in C

This repository contains C programs focused on **Data Structures and Algorithms (DSA)**. Ideal for beginners, college-level coursework, and anyone preparing for technical interviews.

---

## 🔧 What’s Included?

| 📁 Topic             | 📄 Programs Include                              |
|----------------------|--------------------------------------------------|
| 🔸 Linked Lists       | Create, insert, delete, traverse, swap nodes     |
| 🔸 Stacks             | Push, pop, peek using arrays and linked lists    |
| 🔸 Queues             | Enqueue, dequeue, circular queue, priority queue |
| 🔸 Arrays             | Search, sort, insert, delete                     |
| 🔸 Strings            | Reverse, palindrome, compare, manipulation       |
| 🔸 Trees              | Binary Tree, BST, traversals (inorder etc.)      |
| 🔸 Sorting Algorithms | Bubble, Selection, Insertion, Merge, Quick      |
| 🔸 Searching          | Linear Search, Binary Search                    |

---

## 🚀 Getting Started

### ✅ Prerequisites:
- GCC compiler installed (`gcc --version`)

### ▶️ Compile & Run:
```bash
gcc program_name.c -o output
./output
````

Replace `program_name.c` with your desired file.
